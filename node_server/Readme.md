# Instructions

First check if node and npm is installed
```powershell
node -v
npm -v
```

Then open a new VS code window (in file).

Open the folder node_server
then type
```powershell
npm install
```
To install packages.
Current packages are expressjs and nodemon. 
https://expressjs.com/en/starter/hello-world.html


To test run
```powershell
npm test
```
